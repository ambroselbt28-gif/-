安装步骤（3 分钟完成）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 将此整个文件夹复制到 SillyTavern 的 public 目录下

   例如: SillyTavern\public\villain-game\

2. 在 SillyTavern 的世界书或角色卡中添加 iframe

   详细代码请查看: 快速配置代码.txt【1】

3. 配置 AI 系统提示词

   详细代码请查看: 快速配置代码.txt【2】

4. 刷新 SillyTavern 页面，开始游戏！
